Transparencia

El precio sale de una formula, no de una lista

Cada pieza se cotiza sumando el costo real de material, tiempo de corte, perforado, mano de obra y terminacion — mas gastos fijos, margen e IVA. Cambia el precio del acero o tu tiempo de corte, y el catalogo entero se actualiza solo.
